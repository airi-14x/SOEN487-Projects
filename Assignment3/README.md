# Weather App

A simple Flask application that uses Openweathermap API to show weather.

## Requirements

- Python 3.6
- pip3 install Flask
- pip3 install requests
- pip3 install pytz


## Executing the app

- python3 app.py
- Navigate to http://127.0.0.1:5000/
